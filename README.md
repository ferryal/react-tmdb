## React Movie TMDB

> Features:
>
> 1.  List Movie Page (using endpoint top rated)
> 2.  Movie Detail Page. (Display detail movie)
> 3.  Search Movies Autocomplete
> 4.  Filter By Year (Display List Movie by year)

### Tech Stack

1. React
1. Redux
1. NPM
1. Webpack
1. Emotion
1. Eslint
1. Prettier

### Install Package

```
npm install
```

### Setup Development Mode

```
npm run serve
```
